# Isometric Game Box - Aseprite plugin

_Under development_

This is a plugin to generate isometric cubes and boxes in Aseprite. It mainly support 4px wide corners with may be important to some people for game development

## Installation
As any Aseprite plugin, it is really straight forward to install:
- Open your Aseprite scripts folder (`File > Scripts > Open scripts folder`)
- Drop the lua file from this repo
- Rescan your Aseprite scripts (`File > Scripts > Rescan scripts folder` or `F5`)
- Open the plugin (`File > Scripts > Isometric Game Box`)
